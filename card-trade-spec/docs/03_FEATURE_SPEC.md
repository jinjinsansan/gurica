# 03 - 各ページ詳細仕様

## 共通仕様
- 固定ヘッダー（backdrop-blur）、CTAボタン「無料査定はこちら」
- フッター4カラム + 古物商許可表示
- 全ページに generateMetadata() でSEO設定

## 1. トップページ /
- **Hero**: 見出し「リアルもデジタルも。あなたのカードを高価買取」、CTA2つ、フローティングカード装飾
- **カテゴリ紹介**: 4カラム（ポケカ⚡/遊戯王🌀/ワンピ🏴‍☠️/デジタル💎）、ホバーでアクセントライン
- **特徴バー**: gradient-primary背景、実績数値4つ（font-tech）
- **新着商品**: Supabaseから8件取得、ProductCardグリッド
- **お知らせ**: 最新3件、日付+バッジ+タイトル

## 2. 買取価格一覧 /prices
- **タブ**: 全て/ポケカ/遊戯王/ワンピ/その他/デジタル（URLクエリパラメータ）
- **検索**: インクリメンタルサーチ（300msデバウンス）
- **テーブル**: カード名/カテゴリ/レアリティ(バッジ)/美品価格/やや傷あり/傷あり/更新日
- **データ**: Supabase buyback_prices、ISR revalidate=3600
- レアリティバッジ: UR=金, SR=紫, R=緑, C=青

## 3. 買取の流れ /flow
- **4ステップ**: 📝査定申込→📦商品送付→🔍査定→💰お支払い（番号付き丸+接続線）
- **買取方法**: 宅配/店頭/出張の3カード
- **FAQ**: アコーディオン5〜8問
- 下部CTA「無料査定を申し込む」

## 4. オンライン査定 /assessment
- **2カラム**: 左=サイドパネル（gradient背景、メリット3点）、右=フォーム
- **フォーム項目**: 名前(必須)、メール(必須)、電話、カテゴリ(必須)、カード名(必須)、レアリティ、状態(必須)、枚数(必須)、画像(最大5枚)、備考
- **画像**: D&D対応、プレビュー、Supabase Storage保存
- **送信**: API→DB保存→確認メール→/assessment/thanks

## 5. ECショップ /shop
- **一覧**: カテゴリタブ+並び替え、4列グリッド、20件/ページ
- **ProductCard**: 画像/バッジ(NEW,SALE,RARE)/カテゴリ/商品名/レアリティ/価格/カートボタン
- **詳細 /shop/[id]**: 2カラム（画像ギャラリー/情報）、関連商品4件
- **カート**: Zustand+localStorage、数量変更、送料計算（5000円以上無料）
- **注文 /cart/order**: フォーム→API→DB保存→メール→/cart/thanks
- Phase1は銀行振込のみ

## 6. 会社概要 /company
- 2カラム: 法人情報テーブル + 古物商許可カード
- プレースホルダー（{{...}}）は灰色で「確定後に記載」

## 7. 特定商取引法 /legal
- テーブル形式で全項目（販売業者/責任者/所在地/電話/メール/価格/送料/支払方法/引渡時期/返品）

## 8. お問い合わせ /contact
- 2カラム: 連絡先カード3つ + フォーム
- フォーム: 名前/メール/件名/カテゴリ(買取/販売/デジタル/その他)/本文
- 送信→DB保存→自動返信メール

## 9. お知らせ /news
- 一覧: タイトル+日付+カテゴリバッジ、10件/ページ
- 詳細: Markdown本文（react-markdown）、前後ナビ
