# 05 - 法的要件（古物商・特定商取引法）

## 1. 古物商許可に関する表示義務

サイト上の見やすい場所に以下を表示する義務あり（古物営業法施行規則第14条）:
- **許可番号**: 第○○○○○○○○○○○○号
- **氏名又は名称**: {{COMPANY_NAME}}
- **許可を受けた公安委員会**: {{KOBUTSU_COMMISSION}}

### 表示場所
- フッター（全ページ共通）: `古物商許可 {{KOBUTSU_COMMISSION}} 第{{KOBUTSU_NUMBER}}号 {{COMPANY_NAME}}`
- 会社概要ページ: 専用カードで詳細表示
- 許可番号未取得時は「※ 古物商許可申請中」と表示

## 2. 特定商取引法に基づく表記（EC販売に必須）

/legal ページに以下をテーブル形式で表示:
- 販売業者/運営統括責任者/所在地/電話番号/メールアドレス
- 商品の販売価格（各商品ページに記載）
- 商品代金以外の費用（送料、振込手数料）
- 支払方法（銀行振込）
- 支払時期（注文確定後5営業日以内）
- 商品の引渡時期（入金確認後3〜5営業日以内）
- 返品・交換（到着後7日以内、未開封品に限る）
- 返品送料（お客様負担、初期不良は当社負担）

## 3. プライバシーポリシー（/privacy）

記載項目: 個人情報の収集目的/収集する情報の種類/第三者提供/管理体制/Cookie/問い合わせ窓口

## 4. デジタルカード/NFT取扱いの注意

- デジタルカードは「古物」に該当しない可能性あり → 注記を表示
- 「デジタルカードの取引は古物営業法の対象外です」
- NFT取引特有のリスク表示

## 5. 送料設定（仮）

| 販売時 | 条件 |
|--------|------|
| 5,000円以上 | 無料 |
| 5,000円未満 | 全国一律500円（税込） |

| 買取時 | 条件 |
|--------|------|
| 30点以上 | 送料無料 |
| 30点未満 | お客様負担 |
