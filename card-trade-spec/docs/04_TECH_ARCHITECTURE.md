# 04 - 管理画面仕様

## 1. 認証・認可
- Supabase Auth (email+password)
- profiles.role = 'admin' のみ /admin アクセス可
- middleware.ts で認証ガード
- 初期管理者はSupabaseダッシュボードから手動作成

## 2. レイアウト
- 左サイドバー(250px) + 右メインコンテンツ
- モバイル: サイドバーはドロワー化
- メニュー: 📊ダッシュボード/📦商品/💰買取価格/📋査定/🛒注文/📢お知らせ

## 3. ダッシュボード /admin
- KPIカード4つ: 本日の査定/未対応査定/公開商品数/今月の注文
- 直近のアクティビティ: 最新査定5件+最新注文5件

## 4. 商品管理 /admin/products
- **一覧**: 検索+フィルタ+テーブル（サムネ/名前/カテゴリ/レアリティ/価格/在庫/公開状態/操作）20件/ページ
- **登録/編集**: カテゴリ/カード名/レアリティ/状態/販売価格/買取参考価格/在庫数/説明文(Markdown)/画像(最大5枚)/タグ/公開フラグ
- **削除**: ソフトデリート、確認モーダル

## 5. 買取価格管理 /admin/prices
- インライン編集テーブル（カード名/カテゴリ/レアリティ/美品/やや傷あり/傷あり）
- 行の追加・削除
- 一括保存
- CSVインポート: ファイル選択→プレビュー→実行（重複は上書き）

## 6. 査定管理 /admin/assessments
- **一覧**: ステータスフィルタ(new/in_review/quoted/accepted/completed/cancelled)
- **詳細**: 顧客情報/商品情報/画像ギャラリー/ステータス変更/査定金額入力/管理者メモ
- 「査定結果を送信」→Resendメール送信
- ステータスバッジ色: new=青, in_review=黄, quoted=緑, accepted=紫, completed=グレー, cancelled=赤

## 7. 注文管理 /admin/orders
- **一覧**: ステータスフィルタ(pending/paid/shipped/completed/cancelled)
- **詳細**: 顧客情報/配送先/注文商品一覧/合計/ステータス変更/追跡番号入力/メモ

## 8. お知らせ管理 /admin/news
- **一覧**: タイトル/カテゴリ/公開日/状態
- **作成/編集**: タイトル/カテゴリ(info/campaign/maintenance/price_update)/本文(Markdown)/公開日時/公開フラグ
